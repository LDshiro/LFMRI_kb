# Logs

このディレクトリは、作業履歴、レビュー履歴、ChatGPT Library同期、文献投入を記録します。

## Files

| File | Purpose |
|---|---|
| `weekly/` | 週次レビュー |
| `literature-intake-log.csv` | 文献投入ログ |
| `review-log.csv` | KB・論文ノートのレビュー履歴 |
| `chatgpt-library-sync-log.csv` | Libraryからのダウンロード・反映ログ |
| `decision-log.md` | ADRより軽い日常判断メモ |

## Rule

ログは正確さを優先し、過度に整えすぎない。後から見て「何を、いつ、なぜ変えたか」が分かることを重視します。
