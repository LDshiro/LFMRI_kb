# Experiments

低磁場MRIに関する実験、シミュレーション、再構成検証、QA/QC検討の作業領域です。

## Structure

```text
phantom/              ファントム実験プロトコル
reconstruction/       再構成・ノイズ低減・超解像実験
sequence-simulation/  シーケンス・信号・緩和のシミュレーション
notebooks/            解析ノートブック置き場
```

## Rules

- 実験は1テーマ1ディレクトリにする。
- 実験条件、入力、出力、コードバージョンを記録する。
- 結果だけでなく失敗も残す。
- 臨床データを扱う場合は、倫理・匿名化・アクセス権限を確認する。
