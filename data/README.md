# Data Workspace

このディレクトリは、低磁場MRI研究に関連する公開データセット、データカード、派生表を管理します。

## Rules

- 患者画像や個人情報を含むデータはコミットしません。
- 公開データセットであっても、ライセンスと利用条件を確認します。
- 派生表には、作成元、作成日、処理スクリプト、再現手順を記録します。
- 画像データ本体は必要に応じて外部ストレージまたはGit LFSで管理します。

## Structure

```text
public-datasets.md      候補データセット一覧
dataset-cards/          データセットごとの構造化カード
derived-tables/         文献や解析から作成した派生表
```
