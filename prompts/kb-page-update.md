# Prompt: KB Page Update

```text
以下の文献ノートと既存KBページをもとに、KBページの更新案を作成してください。

制約:
- 既存の構造を維持する。
- 新規主張には出典候補を付ける。
- 原典確認が必要な箇所には `needs-source` を付ける。
- 断定が強すぎる文は慎重な表現にする。
- 更新案はMarkdown diff風に、追加・変更・保留を分けて示す。

出力:

## Proposed additions

## Proposed revisions

## Claims requiring verification

## Evidence table updates

## Suggested PR description
```
